# Header Branding Assets

Place optional customer or company logo files here and reference them from public
Control Plane env vars, for example:

```env
NEXT_PUBLIC_ATULYA_CP_BRAND_LOGO_SRC=/branding/customer-mark.svg
NEXT_PUBLIC_ATULYA_CP_BRAND_LOGO_ALT=Customer name
```

Use compact square marks by default. Add a wordmark and set
`NEXT_PUBLIC_ATULYA_CP_BRAND_LOGO_VARIANT=extended` only when the header needs a
wider logo.
